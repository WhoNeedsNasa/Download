Welcome to Who Needs NASA the game.

About this game.

  • This game is suitable for 3-5 year olds as it contains no graphic or violent content. This game allows children to safely interact with ICT's in order to develop necessary skills.

  • The Game is a 2D running based game. The Idea was from the original Super Mario game, and I believe it will be a great start for someone who doesn't have access to a Nintendo console.

The game is still In Alpha stage and will only feature 1 level.
